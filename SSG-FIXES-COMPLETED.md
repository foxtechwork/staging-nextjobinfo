## ‚úÖ 3. Calendar Date Fix for SSG

### Problem:
Calendar was using `new Date()` in `useState` which gets fixed during SSG build, causing static dates.

### Solution (LeftSidebar.tsx):
```typescript
const [selectedDate, setSelectedDate] = useState<Date | undefined>(undefined);

// Set date on client side to avoid SSG static date issue
useEffect(() => {
  setSelectedDate(new Date());
}, []);
```

**Result**: Calendar now always shows current date, even on static pages.
